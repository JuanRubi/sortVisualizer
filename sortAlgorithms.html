Bubble Sort

The bubble sort is one of the three simple sorting algorithms that we are introduce to. The algorithm consists of checking two side by side elements at a time and swapping them if the left element is bigger than the right element. At the end of each pass the largest element will be sorted on the rightmost unsorted index in the array.

Selection Sort

The selection sort algorithm is the second simple sorting algorithm. The algorithm has a pointer that points to the leftmost unsorted index. In each pass the smallest element index is updated and at the end of the array we swap the value at the pointer with the value at the smallest element index.

Insertion Sort

The insertion sort algorithm is the last simple sorting algorithm. For this algorithm we have a pointer that points to the first unsorted element in the array (the second element at the start). We then compare the value at the pointer with the element to its left and swap them if the left element is larger. This comparing is repeated until we encounter an element that is smaller than the current element at hand. The pointer is then shifted to the right once and the process is repeated until our pointer points to the last element.
